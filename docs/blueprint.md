# **App Name**: Red Conectada

## Core Features:

- Interactive Connectivity Mesh: A dynamic Three.js or Vanta.js hero background that visualizes institutional nodes and links responding to mouse movement.
- Asymmetric Video Showcase: A high-performance video carrusel featuring focused scaling and auto-play loops for high-impact visual storytelling.
- Institutional Program Assistant: An AI-powered tool that helps institutions identify synergies and automatically drafts collaboration proposals based on the organization's focus.
- Segmented Impact Sections: Zig-zag modules for 'Ruta de Sonrisas' and 'Enraizando' with immersive glassmorphism and specialized color gradients.
- Dynamic Partner Logo Loop: Infinite monochrome slider of allied organizations that restores color on interaction.
- Glassmorphism Navigation: Intelligent navbar that adapts opacity and backdrop-blur effects based on scroll position.

## Style Guidelines:

- Primary color: Electric Cyan (#00E5FF) for digital energy and focus. Background: Deep Institutional Blue (#001A33), a high-contrast dark theme providing depth. Accent: Vivid Mint (#00FFAB) used sparingly to represent organic growth and sustainability.
- Font pairing: 'Montserrat Bold' for high-impact headlines (H1/H2) and 'Inter' for functional, highly legible body text and navigation. Note: currently only Google Fonts are supported.
- Minimalist linear vector icons with 2px stroke width, consistent with FontAwesome styles for institutional clarity.
- Editorial-style grid with significant negative space, utilizing a zig-zag rhythm in sub-sections to guide user visual flow.
- Seamless transitions including auto-scrolling sliders and smooth focus-scaling for video components.
